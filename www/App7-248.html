◄ ప్రహ్లాదుని జన్మంబు►
7-248-క.
ఆడుదము మనము హరిరతిఁ
బాడుద మే ప్రొద్దు విష్ణుభద్రయశంబుల్
వీడుదము దనుజసంగతిఁ
గూడుదము ముకుందభక్తికోటిన్ సూటిన్.
ప్రతిపదార్ధము
ఆడుదము =ఆడుకొనెదము; మనము = మనము; హరి = విష్ణుని యందలి {హరి - భక్తుల హృదయమునుఆకర్షించువాడు, విష్ణువు}; రతిన్ = ప్రీతిచేత; పాడుదము = పాడుదము; ఏ = అన్ని; ప్రొద్దున్ = వేళనైనను; విష్ణు = హరి; భద్ర = మంగళకరములైన; యశంబుల్ = కీర్తులు; వీడుదము = వదలివేసెదము; దనుజ = రాక్షసులతోడి; సంగతిన్ = చెలిమిని, సాంగత్యమును; ముకుంద = విష్ణుని {ముకుందుడు - ముక్తిని యిచ్చువాడు, విష్ణువు}; భక్తి =భక్తుల; కోటిన్ = సమూహమును; సూటిన్ = సూటిగా.
భావము
మనం శ్రీహరి మీది చిత్తముతో ఆడుకుందాం రండి. మాధవుడిని మనసు నిండా నింపుకుని హరిసంకీర్తనలు పాడుకుందాం. మిగిలిన రాక్షసుల స్నేహం విడిచిపెడదాం. నిర్భయంగా విష్ణుభక్తులతో చేరిపోదాం రండి. ఆనందిద్దాం ఎల్లప్పుడు.
