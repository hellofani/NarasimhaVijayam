7-205-వచనము.

కావున వీఁడు మహాప్రభావసంపన్నుండు వీనికెందును భయంబు లేదు; వీనితోడి విరోధంబునం దనకు మృత్యువు సిద్ధించు" నని నిర్ణయించి చిన్నఁబోయి ఖిన్నుండై ప్రసన్నుండు గాక క్రిందు జూచుచు విషణ్ణుండై చింతనంబు జేయుచున్న రాజునకు మంతనంబునఁ జండామార్కు లిట్లనిరి.

టీకా:-
కావున = అందుచేత; వీడు = ఇతడు; మహా = గొప్ప; ప్రభావ = మహిమయనెడి; సంపన్నుడు = సంపదగలవాడు; వీని =ఇతని; కిన్ = కి; ఎందున్ = ఎక్కను; భయంబు = భయము; లేదు = లేదు; వీని = ఇతని; తోడి = తోటి; విరోధంబునన్ = శత్రుత్వముతో; తన = తన; కున్ = కు; మృత్యవు = మరణము; సిద్ధించును = కలుగును; అని = అని; నిర్ణయించి = నిశ్చయించుకొని; చిన్నబోయి =చిన్నతనముపడి; ఖిన్నుండు = దుఃఖించువాడు; ఐ = అయ్యి; ప్రసన్నుండు =సంతోషముగలవాడు; కాక = కాకుండగ; క్రిందు = కిందకు, నేలచూపులు; చూచుచున్ = చూచుచు; విషణ్ణుండు = విచారపడువాడు; ఐ = అయ్యి; చింతనంబుజేయుచున్న = ఆలోచించుతున్న; రాజున్ = రాజున; కున్ = కు; మంతనంబునన్ = సంప్రదింపులలో; చండామార్కులు =చండామార్కులు; ఇట్లు = ఈ విధముగ; అనిరి = పలికిరి;
బావము:-
ఈ ప్రహ్లాదుడు సామాన్యుడు కాదు, గొప్ప శక్తిమంతుడు. దేనికి భయపడడు.” అని ఆలోచించుకుని, “కనుక ఈ పిల్లాడితో విరోధం పెట్టుకుంటే తనకు మృత్యువు తప్ప” దని నిశ్చయించుకున్నాడు. అనవసరంగా ఈ పసివాడిని పెట్టిన బాధలు తలచుకుని చిన్నబుచ్చుకుంటూ, నేల చూపులు చూస్తూ బాధపడుతున్నాడు దానవేంద్రుడు. అప్పుడు హిరణ్యకశిపుడితో చండామార్కులు ఏకాంతంగా ఇలా ధైర్యం చెప్పారు.
