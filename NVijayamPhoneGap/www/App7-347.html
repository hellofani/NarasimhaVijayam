7-347-ఉత్పలమాల

ప్రాభవ మొప్ప నుత్కటకృపామతియై కదియంగఁ జీరిఁ సం
శోభిత దృష్టిసంఘములఁ జూచుచు బాలుని మౌళి యందు లో
కాభినుతుండు పెట్టె నసురాంతకుఁ డుద్భట కాలసర్ప భీ
తాభయదాన శస్తము ననర్గళ మంగళ హేతు హస్తమున్.

టీకా:-
ప్రాభవము = ఠీవి; ఒప్పన్ = ఒప్పుచుండగా; ఉత్కట = అతిశయించిన; కృపా = కరుణా; మతి = హృదయుడు; ఐ =అయ్యి; కదియంగన్ =దగ్గరకు; చీరి = పిలిచి; సంశోభిత = మిక్కిలి ప్రకాశించెడి; దృష్టిన్ = కన్నుల; సంఘములన్ = సమూహములతో; చూచుచున్ = చూచుచు; బాలుని =పిల్లవాని; మౌళిన్ = తల; అందున్ = పైన; లోకాభిరతుండు = నరసింహస్వామి {లోకాభిరతుడు - లోకములచే అభిరతుడు (కీర్తింపబడువాడు), నరసింహుడు}; పెట్టెన్ =పెట్టెను; అసురాంతకుండు = నరసింహస్వామి {అసురాంతకుడు - అసుర (రాక్షసుని) అంతకుడు (చంపినవాడు), నరహరి}; ఉద్భట = క్రూరమైన; కాల = కాలము యనెడి; సర్పమున్ = పామునకు; భీత = భయపడినవారికి; అభయ = అభయమును; దాన = ఇచ్చెడి; శస్తమున్ =శ్రేష్ఠమైనదానిని; అనర్గళ = అడ్డులేని; మంగళ = శుభములకు; హేతున్ =కారణమైనదానిని; హస్తమున్ = చేతిని.
బావము:-
నరహరి మనసులో కరుణ ఉప్పొంగింది; చిన్న పిల్లవాడు ప్రహ్లాదుడిని దగ్గరకు పిలిచాడు; వాత్యల్యపురస్సరంగా చూస్తూ, ఆ రాక్షస సంహారి తన హస్తంతో మస్తకాన్ని ఆప్యాయంగా నిమిరాడు. ప్రశస్తమైన ఆ హస్తం కాలసర్ప భయాన్ని తొలగించేది; నిత్యమంగళం ప్రసాదించేది.

తెలుగుభాగవతం.ఆర్గ్ సౌజన్యంతో
