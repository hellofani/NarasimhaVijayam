7-229-ఉత్పలమాల

"అంతనిధాన మైన దితిజాధిపువీర్యము దీని కుక్షి న
త్యంత సమృద్ధి నొందెడి మహాత్మక! కావునఁ దత్ప్రసూతి ప
ర్యంతము బద్ధఁ జేసి జనితార్భకు వజ్రము ధారఁ ద్రుంచి ని
శ్చింతుఁడనై తుదిన్ విడుతు సిద్ధము దానవరాజవల్లభన్."

టీకా:-
అంత = నాశనమునకు; నిధానము = కారణమైవవాడు; ఐన = అయిన; దితిజాధిపు = రాక్షసరాజు యొక్క; వీర్యము =రేతస్సు; దీని = ఈమె యొక్క; కుక్షిన్ = కడుపులో; సమృద్ధిన్ = మిగులవృద్ధిని; ఒందెడిన్ = పొందుచున్నది; మహాత్మక = గొప్పఆత్మకలవాడ; కావునన్ = అందుచేత; తత్ =ఆ; ప్రసూతి = పురుటి; పర్యంతమున్ = వరకు; బద్దన్ = బంధీని; చేసి = చేసి; జనిత =పుట్టిన; అర్భకున్ = పిల్లవానిని; వజ్రము = వజ్రాయుధము; ధారన్ = పదునుతో; త్రుంచి = నరికి; నిశ్చింతుడను = దిగులులేనివాడను; ఐ = అయ్యి; తుదిన్ = చివరకు; విడుతున్ = విడిచిపెట్టెదను; సిద్ధము = నిశ్చయముగ; దానవ = రాక్షస; రాజ = రాజుయొక్క; వల్లభన్ = భార్యను.
బావము:-
“ఓ మహాత్ముడా! లోకాలకు దుస్సహమైన హిరణ్యకశిపుని రాక్షస వీర్యం ఈమె కడుపులో వృద్ధి చెందుతూ ఉంది. కాబట్టి ఈమె ప్రసవించే వరకు బందీగా ఉంచి, పుట్టిన బిడ్డను పుట్టినట్లే నా వజ్రాయుధంతో సంహరిస్తాను. అప్పుడు నా మనస్సు నిశ్చింతగా ఉంటుంది. ఈ రాక్షసరాజు పత్నిని తప్పక విడిచిపెట్టేస్తాను.”

తెలుగుభాగవతం.ఆర్గ్ సౌజన్యంతో
