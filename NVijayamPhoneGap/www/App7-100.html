7-100-శార్దూలము.

ఏ దిక్పాలురఁ జూచి నేఁ డలుగునో యే దేవు బంధించునో
యే దిగ్భాగముమీఁద దాడి చనునో యే ప్రాణులం జంపునో
యీ దైత్యేశ్వరుఁ డంచు నొండొరులు దా రింద్రాదు లాస్తాన భూ
వేదిన్ మెల్లన నిక్కి చూతురు భయావిర్భూత రోమాంచు లై.

టీకా:-
ఏ = ఏ; దిక్పాలురన్ = దిక్పాలకులని; చూచి = చూసి; నేడు = ఈ దినమున; అలుగునో = కోపించునో; ఏ = ఏ; దేవున్ = దేవుని; బంధించునో = బంధించివేయునో; ఏ = ఏ; దిగ్భాగము = దిక్కునగల ప్రదేశముల; మీద = పైన; దాడి = యుద్దమునకు; చనునో = వెళ్ళునో; ఏ = ఏ; ప్రాణులన్ = జీవులను; చంపునో = సంహరించునో; ఈ = ఈ; దైత్యేశ్వరుడు = రాక్షసరాజు; అంచున్ = అనుచు; ఒండొరుల = ఒక్కొక్కరు; తారు = వారు; ఇంద్రా = ఇంద్రుడు; ఆదులు = మొదలగువారు; అస్థాన = రాజాస్థానపు సభా; భూవేదిన్ = మండపము మీద; మెల్లన = మెల్లగా; నిక్కి = సాగి; చూతురు = చూచెదరు; భయా = భయముచేత; ఆవిర్భూత = కలిగిన; రోమాంచులు = గగుర్పాటుగలవారు; ఐ = అయ్యి.
