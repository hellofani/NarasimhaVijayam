7-31-చంపకమాల

వనముల నుండుఁ జొచ్చు ముని వర్గములోపల ఘోణిగాఁడు సం
జనన మెఱుంగ రెవ్వరును జాడ యొకింతయు లేదు తన్ను డా
సిన మఱి డాయు వెంటఁబడి చిక్కక చిక్కఁడు వీని నొక్క కీ
లున మన మెల్ల లోఁబడక లోఁబడఁ బట్టుకొనంగవచ్చునే?

టీకా:-
వనములన్ = అడవులందు, నీటియందు; ఉండున్ = ఉండును; చొచ్చున్ = ప్రవేశించును; ముని = మునుల; వర్గము =సమూహముల; లోపల = మనసులలోపల; ఘోణిగాడు = వరాహస్వరూపుడు; సంజననము = అసలు పుట్టుక; ఎఱుంగరు = తెలిసికొనలేరు; ఎవ్వరును = ఎవరూకూడ; జాడ = గుర్తించుటకు; ఒకింతయున్ =కొంచముకూడ; లేదు = అవకాశములేదు; తన్ను = తనను; డాసినన్ = సమీపించెదమంటే; మఱి =మరి; డాయున్ = దగ్గరగును, దాగికొనును; వెంటబడి = వెనుకనంటినను; చిక్కక =ఆశ్రయించక, దొరకనే; చిక్కడు = దొరకడు; వీనిన్ = ఇతనిని; ఒక్క = ఒక; కీలునన్ =ఉపాయముచే, యుక్తిచే; మనము = మనము; ఎల్లన్ = అందరము; లోబడక = అతనికివశముగాక, వెనుదీయక; లోబడన్ = చిక్కునట్లు; పట్టుకొనంగ = పట్టుకొనుట; వచ్చునే = సాధ్యమాఏమి.
బావము:-
ఆ సూకరం గాడు మహా మాయగాడు. నీళ్ళల్లో దాక్కుంటాడు. అడవులలో దాక్కుంటాడు. మునుల మనసులలో దూరతాడు. పుట్టుపూర్వోత్తరాలు ఏవీ ఎవరికీ తెలియవు. అసలు ఎక్కడివాడో కూడా తెలియదు. దరిచేరిన వారిని ఆదరిస్తాడట. కాని వెంటబడి పట్టుకుందాం అంటే దొరకనే దొరకడు, అలసిపోడు. కాబట్టి మనం ఒక ఉపాయం పన్నాలి. మనం వాడికి లొంగకుండా వాడిని లొంగదీసుకుందాం, సరేనా? (ఇది సాధ్యమా?)

తెలుగుభాగవతం.ఆర్గ్ సౌజన్యంతో
