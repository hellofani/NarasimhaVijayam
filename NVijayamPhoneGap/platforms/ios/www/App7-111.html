7-111-ఆటవెలది.

శుద్ధసాధు లందు, సుర లందు, శ్రుతు లందు, 
గోవు లందు, విప్రకోటి యందు, 
ధర్మపదవి యందుఁ దగిలి నా యందు వాఁ
డెన్నఁ డలుగు నాఁడె హింస నొందు.

టీకా:-
శుద్ధ = స్వచ్ఛమైన; సాధులు = సజ్జనుల; అందున్ = లోను; సురలు = దేవతలు; అందున్ = లోను; శ్రుతుల = వేదముల; అందున్ = లోను; గోవుల = గోవుల; అందున్ = లోను; విప్ర = బ్రాహ్మణుల; కోటి = సమూహము; అందున్ = లోను; ధర్మ = ధర్మయొక్క; పదవి = మార్గము; అందున్ = లోను; తగిలి = పూని; నా = నా; అందున్ = ఎడల; వాడు = వాడు; ఎన్నడు = ఏప్పుడైతే; అలుగు = కోపగించునో; నాడె = ఆ దినముననే; హింసన్ = మరణమును; ఒందున్ = పొందును.
బావము:-
కపటములేని సాధుజనులు తోను, దేవతల తోను, వేదాల తోను, గోవుల తోను, బ్రాహ్మణుల తోను, ధర్మతోను, నాతోను ఎప్పుడైతే పట్టుబట్టి శత్రుత్వం వహిస్తాడో అప్పుడే వాడు చస్తాడు.

తెలుగుభాగవతం.ఆర్గ్ సౌజన్యంతో
