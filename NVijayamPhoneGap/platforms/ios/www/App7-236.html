7-236-ఆటవెలది.

వినుఁడు నాదు పలుకు విశ్వసించితిరేని
సతుల కయిన బాల జనుల కయినఁ
దెలియ వచ్చు మేలు దేహాద్యహంకార
దళననిపుణ మైన తపసిమతము."

టీకా:-
వినుడు = వినండి; నాదు = నా యొక్క; పలుకు = మాట; విశ్వసించితిరేని = నమ్మినచో; సతుల్ = స్త్రీల; కున్ = కు; అయిన = ఐన; బాల = పిల్ల; జనుల్ = వారి; కిన్ = కి; అయిన = ఐన; తెలియన్ = తెలియను; వచ్చున్ = అగును; మేలు = మంచిది, శ్రేయము; దేహ = దేహము; ఆది = మొదలగు; అహంకార = మమత్వమును; దళన = పోగొట్టునది; ఐన = అయిన; తపసి = ఋషిచే; మతము = తెలుపబడినది.
బావము:-
శ్రద్దగా వినండి చెప్తాను. నా మాట నమ్మండి. నారద మహర్షి తత్వం తెలుసుకోడానికి స్త్రీలు, బాలలు కూడ అర్హులే. ఈ నారద భక్తి తత్వం తెలుసుకుంటే దేహాభిమానాలు, మమకారాలు తొలగిపోతాయి. ఉత్తమమైన భక్తి ఏర్పడుతుంది.”

తెలుగుభాగవతం.ఆర్గ్ సౌజన్యంతో
