7-66-కంద పద్యము.

కాలము వచ్చిన శబరుని
కోలను ధరఁగూలె ఖగము ఘోషముతోడం
గాలము డాసిన నేలం
గూలక పో వశమె యెట్టి గుణవంతులకున్.

టీకా:-
కాలము = అదను, సమయము; వచ్చిన = రాగా; శబరుని = కోయవాని; కోలను = బాణమువలన; ధరన్ = నేలపైన; కూలెన్ = పడిపోయెను; ఖగము = పక్షి; ఘోషము = గోలపెట్టుట; తోడన్ = తోటి; కాలము =కాలము, అదను; డాసిన = కలిసివచ్చిన; నేలంగూలక = మరణించకుండుట; వశమె = సాధ్యమేనాఏమి; ఎట్టి = ఎటువంటి; గుణవంతుల = తెలివిగలవారి; కున్ = కైనను.
బావము:-
కాలం మూడిన ఆ మగ పక్షి గోలపెట్టిమరీ అలా బోయవాడి బాణం దెబ్బతిని పడి చచ్పిపోయింది. అవును, ఆయువు తీరిన తరువాత చావకుండా ఉండటం అన్నది ఎంతటి తెలివైనవాడైనా, బలవంతుడైనా, అధికార సంపదలు కలవాడి కైనా అసాధ్యమే కదా.

తెలుగుభాగవతం.ఆర్గ్ సౌజన్యంతో
