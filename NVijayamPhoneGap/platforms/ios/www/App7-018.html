7-18-శార్దూలము

కామోత్కంఠత గోపికల్, భయమునం గంసుండు, వైరక్రియా
సామాగ్రిన్ శిశుపాలముఖ్య నృపతుల్, సంబంధులై వృష్ణులున్, 
బ్రేమన్ మీరలు, భక్తి నేము, నిదె చక్రింగంటి; మెట్లైన ను
ద్ధామ ధ్యానగరిష్ఠుఁ డైన హరిఁ జెందన్ వచ్చు ధాత్రీశ్వరా!

టీకా:-
కామ = మన్మథ వికారమునందలి; ఉత్కంఠతన్ = ఆసక్తితో; గోపికల్ = గోపికలు; భయమునన్ = భయముతో; కంసుండు =కంసుడు; వైర = విరోధపు; క్రియన్ = చేతల; సామగ్రిన్ = బలమున; శిశుపాల =శిశుపాలుడు; ముఖ్య = మున్నగు; నృపతుల్ = రాజులు; సంబంధులు = చుట్టరికము గల వారు; ఐ = అయ్యి; వృష్ణులున్ = యాదవులు {వృష్ణులు - వృష్ణవంశపు యాదవులు}; ప్రేమన్ =ప్రేమతో; మీరలు = మీరు; భక్తిన్ = భక్తితో; ఏమున్ = మేము; ఇదె = ఇదిగో; చక్రిన్ = విష్ణుమూర్తిని {చక్రి – చక్రము ధరించిన వాడు, హరి}; కంటిమి = దర్శించితిమి; ఎట్లు = ఏ విధముగ; ఐనన్ = అయినప్పటికిని; ఉద్దామ = ఉన్నత మైన; ధ్యాన =ధ్యానముచే; గరిష్ఠుడు = శ్రేష్ఠుడు; ఐన = అయిన; హరిన్ = నారాయణుని; చెందన్ =పొంద; వచ్చు = సాధ్యము; ధాత్రీశ్వరా =రాజా {ధాత్రీశ్వరుడు - ధాత్రి (భూమి)కిఈశ్వరుడు (ప్రభువు), రాజు}.
బావము:-
ధర్మరాజా! మన్మథ వికారముతో గోపికలు, ప్రాణభయంతో కంసుడు, విరోధంతో శిశుపాలుడు మొదలైన రాజులు, దాయాదులు అయ్యి యాదవులు, బంధుప్రీతితో మీరు, భక్తితో మేము నిరంతర స్మరణలు చేసి విష్ణుసాయుజ్యం పొందాము కదా. ఏ విధంగా అయినా సరే విడవకుండా ధ్యానించి నారాయణుని పొందవచ్చును.

తెలుగుభాగవతం.ఆర్గ్ సౌజన్యంతో
