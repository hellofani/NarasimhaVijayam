◄ బ్రహ్మవరములిచ్చుట►
7-86-వ.
ఇట్లు దైత్యేశ్వరుండు దివి నున్న హంసవాహనునకు ధరణీతలంబున దండప్రణామంబు లాచరించి సంతోషబాష్పసలిలసంవర్ధిత పులకాంకురుండై యంజలి జేసి కంజాతగర్భుని మీఁద దృష్టి యిడి గద్గదస్వరంబున నిట్లని వినుతించె.
ప్రతిపదార్ధము
ఇట్లు = ఈ విధముగ; దైత్యేశ్వరుండు = హిరణ్యకశిపుడు {దైత్యేశ్వరుడు - దైత్య (రాక్షసుల)కు ఈశ్వరుండు(ప్రభువు),హిరణ్యకశిపుడు}; దివిన్ = ఆకాశమున; ఉన్న = ఉన్నట్టి; హంసవాహనున్ = బ్రహ్మదేవుని {హంసవాహనుడు - హంస వాహనముగా కలవాడు, బ్రహ్మ}; కున్ = కి; ధరణీతలంబునన్ = నేలపైన; దండప్రణామంబులు = దండవన్నమస్కారములు {దండప్రణామము - కర్రవలె సాగి మ్రొక్కుట}; ఆచరించి = చేసి; సంతోష = ఆనందపు; బాష్పసలిల = కన్నీటిచే; సంవర్ధిత = అతిశయించిన; పులకాంకురుండు = గగుర్పాటుగలవాడు; ఐ = అయ్యి; అంజలి = నమస్కారము; చేసి = పెట్టి; కంజాతగర్భుని = బ్రహ్మదేవుని {కంజాకగర్భుడు - కంజాత (నీటపుట్టినది పద్మము)న గర్భుడు(పుట్టినవాడు), బ్రహ్మ}; మీదన్ = పైన; దృష్టి = చూపు; ఇడి = పెట్టి; గద్గదస్వరంబునన్ = డగ్గుతికతో; ఇట్లు = ఈ విధముగ; అని = పలికి; వినుతించె = స్తోత్రము చేసెను.
భావము
ఇలా ఆ హిరణ్యకశిప రాక్షస రాజు ఆకాశంలో హంసవాహనుడై ఉన్న బ్రహ్మదేవుడికి నేలమీద పడిపడి నమస్కారాలు చేసాడు. ఆనంద భాష్పాలతో, పులకరించిన ఒంటితో అంజలి ఘటించి నమస్కారం చేసి, కమలాసనుపైనే దృష్టి నిలుపుకొని బొంగురుపోయిన కంఠంతో ఇలా స్తుతించసాగాడు.
