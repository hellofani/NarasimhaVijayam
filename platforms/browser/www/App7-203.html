◄ ప్రహ్లాదుని హింసించుట►
7-203-వ.
అదియునుం గాక తొల్లి శునశ్శేఫుం డను మునికుమారుండు దండ్రి చేత యాగపశుత్వంబునకు దత్తుం డయి తండ్రి తనకు నపకారి యని తలంపక బ్రదికిన చందంబున.
ప్రతిపదార్ధము
అదియునున్ = అంతే; కాక = కాకుండగ; తొల్లి = పూర్వము; శునశ్శేపుండు = శునశ్శేపుడు; అను = అనెడి; ముని = మునుల; కుమారుండు = పుత్రుడు; తండ్రి = తండ్రి; చేతన్ = వలన; యాగ =యజ్ఞమునకైన; పశుత్వంబున్ = బలిపశువు అగుట; కున్ = కు; దత్తుండు = ఇయ్యబడినవాడు; అయి = అయ్యి; తండ్రిన్ = తండ్రిని; తన = తన; కున్ = కు; అపకారి = హానిచేసినవాడు; అని = అని; తలపకన్ = భావించకుండగ; బ్రదికిన = ఆపదనుండి బయటపడిన; చందంబునన్ = విధముగ.
భావము
పూర్వకాలంలో ఒక ముని ఉండేవాడు. అతని శునశ్శేపుడు అనే కొడుకు ఉండే వాడు. ఆ తండ్రి ధనం కోసం తన కొడుకును యాగ పశువుగా ఇచ్చేసేడట. అయినా కూడ ఆ బాలుడు తండ్రిని అపకారిగా భావించకుండా ఉపకారిగానే భావించి జీవించి ఉన్నాడట. అలాగే నా కొడుకు ప్రహ్లాదుడు కూడ ఉన్నాడే!
