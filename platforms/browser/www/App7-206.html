◄ ప్రహ్లాదుని హింసించుట►
7-206-శా.
"శుభ్రఖ్యాతివి నీ ప్రతాపము మహాచోద్యంబు దైత్యేంద్ర! రో
షభ్రూయుగ్మ విజృంభణంబున దిగీశవ్రాతముం బోరులన్
విభ్రాంతంబుగఁ జేసి యేలితి గదా విశ్వంబు వీఁ డెంత? యీ
దభ్రోక్తుల్ గుణదోషహేతువులు చింతం  బొంద నీ కేటికిన్?
ప్రతిపదార్ధము
శుభ్ర =పరిశుద్ధమైన; ఖ్యాతివి = కీర్తిగలవాడవు; నీ = నీ యొక్క; ప్రతాపము = పరాక్రమము; మహా = గొప్ప; చోద్యము = చిత్రము; దైత్యేంద్ర = రాక్షసరాజ; రోష = క్రోధముతోకూడిన; భ్రూ = కనుబొమల; యుగ్మ = జంట యొక్క; విజృంభణంబునన్ = విప్పారుటవలన; దిగీశ =దిక్పాలకుల {దిక్పాలురు - 1ఇంద్రుడు 2అగ్ని 3యముడు 4నిరృతి 5వరుణుడు 6వాయువు 7కుబేరుడు 8ఈశానుడు}; వ్రాతము = సమూహమునుకూడ; విభ్రాజితంబుగన్ = కలతపడినదిగ; చేసి = చేసి; ఏలితి = పాలించితివి; కదా = కదా; విశ్వంబున్ = జగత్తును; వీడు =ఇతడు; ఎంత = ఏపాటివాడు; ఈ = ఇట్టి; దభ్ర = అల్పపు; ఉక్తులు = పలుకులు; గుణ =సుగుణములకు; దోష = దోషమునుకలుగుటకు; హేతువులు = కారణములు; చింతన్ = విచారమున; పొందన్ = పడుట; నీ = నీ; కున్ = కు; ఏటికిన్ = ఎందుకు.
భావము
“ఓ రాక్షసేంద్రా! నీవు నిర్మలమైన కీర్తిశాలివి. నీ ప్రతాపం అత్యద్భుతమైనది. నీవు యుద్ధంలో ఒక మాటు కనుబొమ్మలు కోపంతో చిట్లిస్తే చాలు దిక్పాలకులు సైతం భయపడిపోతారు. ఇలా ప్రపంచం అంతా ఏకఛత్రాధిపత్యంగా ఏలావు. అంతటి నీకు పసివాడు అనగా ఎంత? ఈ మాత్రానికే ఎందుకు విచారపడతావు? ఇంతకు ఇతడు పలికే తెలిసీ తెలియని మాటలకు నువ్వు దిగులు పడటం దేనికి?” అని చెప్తూ చండామార్కులు మంచి గుణాలున్న మంచి మాటలు, చెడ్డ గుణములున్న చెడ్డ మాటలు నని యెఱుఁగవా? మంచి బుద్ధి గలిగినవాఁడే బాగుపడును. లేకున్న చెడిపోవును. నీ కేల వృధాప్రయాసము? అని సమాధాన పరుస్తూ, చండామార్కులు ఇంకా ఇలా చెప్పసాగారు.
