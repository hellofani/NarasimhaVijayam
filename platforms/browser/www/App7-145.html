7-145-శా.
నాకుం జూడఁగఁ జోద్య మయ్యెడిఁ గదా నా  తండ్రి! యీ బుద్ధి దా
నీకున్ లోపలఁ దోఁచెనో? పరులు దుర్నీతుల్ పఠింపించిరో? 
యేకాంతంబున భార్గవుల్ పలికిరో? యీ  దానవశ్రేణికిన్
వైకుంఠుండు గృతాపరాధుఁ డతనిన్ వర్ణింప నీ కేటికిన్?
ప్రతిపదార్ధము
నా = నా; కున్ = కు; చూడగన్ = చూచుటకు; చోద్యము = చిత్రము; అయ్యెడిగదా = కలుగుతున్నది; నా = నాయొక్క; తండ్రి = నాయనా; ఈ = ఇట్టి; బుద్ధి = భావము; తాన్ = దానంతటదే; నీ = నీ; కున్ = కు; లోపలన్ = మనసునందు; తోచెనో = కలిగినదా లేక; పరులు = ఇతరులు; దుర్నీతుల్ = చెడ్డవారు; పఠింపించిరో = చదివించిరా లేక; ఏకాంతమునన్ = రహస్యమున; భార్గవుల్ = చండామార్కులు {భార్గవులు - భర్గుని (శుక్రుని) కొడుకులు, చండామార్కులు}; పలికిరో = చెప్పిరా ఏమి; ఈ = ఈ; దానవ = రాక్షసుల; శ్రేణి =కులమున; కిన్ = కు; వైకుంఠుడు = నారాయణుడు {వైకుంఠుడు - వైకుంఠమున ఉండువాడు, విష్ణువు}; కృత = ఒనర్చిన; అపరాధుడు = ద్రోహముగలవాడు; అతనిన్ = అతనిని; వర్ణింపన్ = స్తుతించుట; నీ = నీ; కున్ = కు; ఏటికిన్ = ఎందులకు.
భావము
ఓ నా కుమారా! ప్రహ్లాదా! చూస్తుంటే ఇదంతా నాకు వింతగా ఉంది. ఇలాంటి బుద్ధి నీ అంతట నీకే కలిగిందా? లేక పరాయి వాళ్ళు ఎవరైనా ఎక్కించారా? లేక నీ గురువులు రహస్యంగా నేర్పారా? విష్ణువు మన రాక్షసులకు ఎంతో ద్రోహం చేసినవాడు. అతనిని కీర్తించకు, అతని పేరు కూడా తలచుకోకు.
