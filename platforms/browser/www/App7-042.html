7-42-శా.
స్రస్తాకంపిత కేశబంధములతో సంఛిన్నహారాళితో
హస్తాబ్జంబులు సాఁచి మోఁదికొనుచున్ హా  నాథ! యంచున్ బహు
ప్రస్తావోక్తులతోడ నేడ్చిరి వగం బ్రాణేశు పాదంబుపై
నస్తోక స్తనకుంకమారుణ వికీర్ణాస్రంబు వర్షించుచున్.
ప్రతిపదార్ధము
స్రస్తా = జారిపోయి; ఆకంపిత = పొరలుచున్న; కేశబంధముల్ = కొప్పులు; తోన్ = తోటి; సంఛిన్న = తెగిపోయిన; హార = హారముల; ఆవళి = సమూహము; తోన్ = తోటి; హస్తా = చేతులు యనెడి; అబ్జంబులు =పద్మములు; సాచి = చాచి; మోదికొనుచున్ = రొమ్ముపైకొట్టుకొనుచు; హా = అయ్యో; నాథ =మగడా; అంచున్ = అనుచు; బహు = అనేకమైన; ప్రస్తావ = ప్రస్తావించెడి; ఉక్తుల =పలుకులు; తోడన్ = తోటి; ఏడ్చిరి = దుఃఖించిరి; వగన్ = శోకముతో; ప్రాణేశు = మగని {ప్రాణేశుడు - ప్రాణములకు ఈశుడు (ప్రభువు), భర్త}; పాదాల = పాదముల; పైన్ = మీద; అస్తోక = విస్తారమైన; స్తన = చనులయందలి; కుంకుమ = కుంకుమచే; అరుణ = ఎర్రబారి; వికీర్ణ = జారుతున్న; అస్రంబున్ = కన్నీళ్ళను; వర్షించుచున్ = కురియుచు.
భావము
ఆ రాజు కాంతలు తలలు బాదుకుని ఏడుస్తున్నారు. వారి సుందరమైన కేశ బంధాలు జారిపోయి చిందర వందరలు అయిపోయాయి. కంఠాలలోని హారాలు తెగిపోయాయి. చేతులు చాచి గుండెలు బాదుకుంటూ “నాథా! హా నాథా!” అంటూ ఆర్తనాదాలు చేస్తున్నారు. ఆపుకోలేని దుఃఖంతో భర్త పాదాలపై పడి గత స్మృతుల తలచుకుంటూ విలపిస్తున్నారు. కన్నీరు కాలువలు కట్టి పాలిండ్లపై పడి ప్రాకి కుంకుమ లేపనం కరిగి ప్రవహిస్తుండగా, కన్నీరు మున్నీరుగా రోదిస్తున్నారు.
