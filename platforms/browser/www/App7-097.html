◄ బ్రహ్మవరములిచ్చుట►
7-97-క.
కృశలై సంప్రాపిత దు
ర్దశలై సర్వంకషోగ్ర దైతేయాజ్ఞా
వశలై నిశలును దివములు
దిశ లెల్లను గట్టుబడియె దీనత ననఘా!
ప్రతిపదార్ధము
కృశలు = కృశించినవి; ఐ = అయ్యి; సంప్రాపిత = పొందింపబడిన; దుర్దశలు = హీనమైన స్థితులనుగలవి; ఐ = అయ్యి; సర్వ = అందరిని; అంకష = వత్తడిచేయుచున్న; ఉగ్ర = భయంకరమైన; దైతేయ = రాక్షసుని; ఆజ్ఞా = ఉత్తరములకు; వశలు = లోనైనవి; ఐ = అయ్యి; నిశలున్ = రాత్రులు; దివముల్ = పగళ్ళు; దిశలు = దిక్కులు; ఎల్లను = సమస్తమును; కట్టుబడియెన్ = కట్టుబడిపోయెను; దీనతన్ = జాలికిలోనై; అనఘా = పుణ్యుడా.
భావము
ఓ పుణ్యాత్ముడైన ధర్మరాజు! దిక్కులన్నీ రాత్రింబవళ్ళు కృంశించి, ఎల్లెడలా రాక్షసుల ధిక్కారాలకు లోనౌతూ ఎంతో దీనావస్థలను పొందుతూ ఉన్నాయి.
