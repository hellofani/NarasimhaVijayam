◄ ప్రహ్లాద చరిత్రము►
7-142-ఉ.
ఎల్ల శరీరధారులకు నిల్లను చీఁకటినూతిలోపలం
ద్రెళ్ళక వీరు నే మను మతిభ్రమణంబున భిన్ను లై ప్రవ
ర్తిల్లక సర్వము న్నతని దివ్యకళామయ మంచు విష్ణునం
దుల్లముఁ జేర్చి తా రడవి నుండుట మేలు నిశాచరాగ్రణీ!
ప్రతిపదార్ధము
ఎల్ల = సర్వ; శరీరధారుల్ = మానవుల {శరీరధారులు - దేహము ధరించినవారు, మానవులు}; కున్ = కు; ఇల్లు = నివాసము; అను = అనెడి; చీకటి = చీకటి; నూతి = నుయ్యికి; లోపలన్ =లోపలందు; త్రెళ్ళక = పడకుండగ; వీరున్ = వీళ్ళు; ఏమున్ = మేము; అను = అనెడి; మతిన్ = చిత్త; భ్రమణంబునన్ = వైకల్యముతో; భిన్నులు = భేదభావముగలవారు; ఐ =అయ్యి; ప్రవర్తిల్లక = తిరుగకుండగ; సర్వమున్ = అఖిలము; అతని = అతని యొక్క; దివ్య = అతిగొప్ప; కళా = అంశతో, మాయావిలాసముతో; మయము = నిండినది; అంచున్ = అనుచు; విష్ణున్ = నారాయణుని; అందున్ = అందు; ఉల్లమున్ = హృదయము; చేర్చి = చేర్చి; తారు = తాము; అడవిన్ = అడవిలో; ఉండుట = ఉండుట; మేలు = ఉత్తమము; నిశాచర = రాక్షసులలో; అగ్రణీ = గొప్పవాడ.
భావము
“ఓ రాక్షసేశ్వరా! లోకులు అందరు అజ్ఞానంతో, ఇల్లనే చీకటిగోతిలో పడి తల్లడిల్లుతూ ఉంటారు; “నేను వేరు, ఇతరులు వేరు” అనే చిత్త భ్రమ భేద భావంతో ఉంటారు. అట్టి భేద భావంతో మెలగకుండా; విశ్వం అంతా విష్ణు దేవుని లీలా విశేషాలతో నిండి ఉంది అని గ్రహించాలి; అలా గ్రహించి ఆ విష్ణుదేవుని మనసులో నిలుపుకొని, తాము అడవులలో నివసించినా ఉత్తమమే.”
తెలుగుభాగవతం.ఆర్గ్ సౌజన్యంతో
