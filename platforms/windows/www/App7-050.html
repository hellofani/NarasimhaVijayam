◄ సుయజ్ఞోపాఖ్యానము►
7-50-క.
కలుగును మఱి లేకుండును
గల భూతము లెల్లఁ గాలకర్మవశము లై
నిలుఁవడు ప్రకృతిం దద్గుణ
కలితుఁడు గాఁ డాత్మమయుఁ డగమ్యుఁడు దలపన్.
ప్రతిపదార్ధము
కలుగును = పుట్టును; మఱి = మరల; లేకుండును = నశించును; కల = ఉన్నట్టి; భూతములు = జీవులు; ఎల్లన్ =అన్నియును; కాల = కాలమునకు; కర్మ = కర్మమునకు; వశములు = లోబడినవి; ఐ = అయ్యి; నిలువడు = ప్రతిష్టింపడును; ప్రకృతిన్ = ప్రకృతిలో; తత్ = ఆ దేహము యొక్క; గుణ =గుణములతో; కలితుఁడు = కూడినవాడు; కాడు = అవ్వడు; ఆత్మ = ఆత్మ; మయుడు =స్వరూపుడు; అగమ్యుడు = పొందరానివాడు; తలపన్ = విచారించగా.
భావము
ఆయన నియమించినట్లు ప్రకృతిలో, త్రిగుణాలతో కూడి జీవులన్నీ కాలానికి, కర్మలకు వశమై పుడుతూ, చస్తూ ఉంటాయి. కాని భగవంతుడు ప్రకృతికి త్రిగుణాలకి లొంగి ఉండడు. ఆయన త్రిగుణాలు కలవాడు కాడు, త్రిగుణాలకి అతీతుడు. సమస్తమందు ఆత్మరూపంలో నిండి ఉంటాడు. ఆ పరమాత్మ తత్వం ఎవరికి అర్థం గాదు.
