సభలందును మఖము లందు జగదీశ్వర! నీ
శుభగీతములు పఠించుచు
నభయులమై సంచరింతు మార్తశరణ్యా!
ప్రతిపదార్ధము
త్రిభువన =ముల్లోకములకు; శత్రుడు = శత్రువు; పడియెన్ = చనిపోయెను; సభలు = సభలు; అందును =లోను; మఖములు = యజ్ఞములు; అందున్ = లోను; జగదీశ్వర = నరసింహుడా {జగదీశ్వరుడు -జగత్ (భువనములకు) ఈశ్వరుడు, విష్ణువు}; నీ = నీ యొక్క; శుభ = మంగళ; గీతములున్ =స్తోత్రములను; పఠించుచున్ = పాడుచు; అభయులము = భయములేనివారము; ఐ = అయ్యి; సంచరింతుము = తిరిగెదము; ఆర్తశరణ్య = నరసింహుడా {ఆర్తశరణ్యుడు - ఆర్తులైనవారికిశరణు యిచ్చువాడు, విష్ణువు}.
భావము
“ఓ సకల భువన పాలకా! నరకేసరి! నీవు ఆర్తులకు శరణు ఇచ్చువాడవు. ముల్లోకాలకూ శత్రువు అయిన హిరణ్యకశిపుడు మరణించాడు. ఇంక యజ్ఞశాలలలోనూ, సభావేదికలమీదా నీ వీరగాథలు, యశోగీతాలు గానం చేస్తాం. నిర్భయంగా మా కర్తవ్యం మేము నిర్వహిస్తాము.”
