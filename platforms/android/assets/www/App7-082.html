◄ బ్రహ్మవరములిచ్చుట►
7-82-శా.
దంశవ్రాతములుం బిపీలికలు మేదః  క్రవ్య రక్తంబులన్
సంశీర్ణంబులు జేసి పట్టితినఁగా శల్యావశిష్టుండ వై
వంశచ్ఛన్న తృణావళీ యుత మహావల్మీకమం దింద్రియ
భ్రంశం బింతయు లేక నీకు నిలువం బ్రాణంబు లెట్లుండెనో?
ప్రతిపదార్ధము
దంశ = అడవియీగల; వ్రాతములున్ = గుంపులు; పిపీలికలు = చీమలు; మేదస్ = కొవ్వు; క్రవ్య = కండలు; రక్తంబులన్ = రక్తములను; సంకీర్ణంబులు జేసి = చీల్చి; పట్టి = కొరికి; తినగా = తినివేయగా; శల్య = ఎముకలు; అవశిష్టుండవు = మాత్రము మిగిలినవాడవు; ఐ = అయ్యి; వంశ = వెదురుపొదలుతో; చ్ఛన్న = కప్పబడిన; తృణ = గడ్డి; ఆవళీ = దుబ్బులుతో; యుత = కూడిన; మహా = పెద్ద; వల్మీకము = పుట్ట; అందున్ = లో; ఇంద్రియ = ఇంద్రియములు; భ్రంశంబు = చలించిపోవుట, చెదరుట; లేక = లేకుండగ; నీకు = నీకు; నిలువన్ = నిలబడుటకు; ప్రాణంబుల్ = ప్రాణములు; ఎట్లు = ఏ విధముగ; ఉండెనో = ఉన్నవోకదా.
భావము
ఈగలు, చీమలు పట్టి చీల్చి, చీము, నెత్తురు, కండలు తినేయటంతో ఉట్టి ఎముకల పోగులా మిగిలి ఉన్నావు. నీ మీద వెదురు పొదలు, గడ్డి దుబ్బులు కప్పేసిన పుట్ట పెరిగిపోయి నువ్వు కనబడటంలేదు. అయినా ఇంద్రియాలను నిగ్రహించుకొని తపస్సు చేస్తూనే ఉన్నావు. విచిత్రంగా ఇలా నువ్వు ఇన్నేళ్ళు ఎలా జీవించి ఉన్నావయ్యా! హిరణ్యకశిపా!
