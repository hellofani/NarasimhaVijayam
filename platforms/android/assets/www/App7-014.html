◄ నారాయణునవైషమ్యాభావం►
7-14-ఆ.
అలుక నైనఁ జెలిమి నైనఁ గామంబున
నైన బాంధవమున నైన భీతి
నైనఁ దగిలి తలఁప నఖిలాత్ముఁ డగు హరిఁ
జేర వచ్చు వేఱు జేయఁ డతఁడు.
ప్రతిపదార్ధము
అలుకన్ = కోపముతో; ఐనన్ = అయినను; చెలిమిన్ = స్నేహముతో; ఐనన్ = అయినను; కామంబునన్ = కోరికతో; ఐనన్ = అయినను; బాంధవమునన్ = చుట్టరికముతో; ఐనన్ = అయినను; భీతిన్ = భయముతో; ఐనన్ =అయినను; తగిలి = పూని; తలపన్ = తరచిచూసినచో; అఖిలాత్ముడు = సర్వభూతస్వరూపుడు; అగు = అయిన; హరిన్ = నారాయణుని; చేరవచ్చును = చేరుట సాధ్యము; వేఱుచేయడు =భేదముగచూడడు; అతడు = అతడు.
భావము
ఆ స్వామి ఎవరినీ ఎప్పుడూ పరునిగా చూడడు. కోపంతో కానీ, స్నేహంతో కానీ, ఇష్టంతో కానీ, బంధుత్వంతో కానీ, భయంతో కానీ ఎలా అయినా సరే మనస్ఫూర్తిగా ఏకాగ్రంగా విష్ణునామాన్ని స్మరణ చేస్తే చాలు, సర్వ భూతాత్మకు డైన ఆ శ్రీమహావిష్ణువు సన్నిధానానికి చేర్చుకుంటాడు.
