7-314-శా.
చండక్రోధముతోడ దైత్యుఁడు వడిన్ శ్రాద్ధంబులన్ మత్సుతుల్
పిండంబుల్ సతిలోదకంబులుగ నర్పింపంగ మా కీక యు
ద్దండత్వంబునఁ దాన కైకొను మహోదగ్రుండు; వీఁ డిక్కడన్
ఖండింపంబడె నీ నఖంబుల నుతుల్ గావింతు మాత్మేశ్వరా!
ప్రతిపదార్ధము
చండ = తీవ్రమైన; క్రోధము = కోపము; తోడన్ = తోటి; దైత్యుడు = రాక్షసుడు; వడిన్ = వేగముగ; శ్రాద్ధంబులన్ = ఆబ్దికములాది పితృకర్మలలో; మత్ = మా యొక్క; సుతుల్ = సంతతివారు; పిండంబులన్ = పిండములను; సతిలోదకంబులుగన్ = నువ్వులు నీళ్ళతో కూడినవిగా; అర్పింపంగన్ = సమర్పించుతుండగా; మా = మా; కున్ = కు; ఈక = ఈయకుండ; ఉద్ధండత్వంబునన్ = దాష్టీకముతో; తాన = తనే; కైకొనున్ = స్వీకరించును; మహా =మిక్కలి; ఉదగ్రుండు = క్రూరుడు; వీడు = ఇతడు; ఇక్కడన్ = ఇక్కడ; ఖండింపంబడె =చంపబడెను; నీ = నీ; నఖంబులన్ = గోళ్ళతోటి; నుతుల్ = స్తోత్రములను; కావింతుము =చేసెదము; ఆత్మేశ్వరా = నరసింహుడా {ఆత్మేశ్వరుడు - ఆత్మ (తమయొక్క) ఈశ్వరుడు (దేవుడు), విష్ణువు}.
భావము
“ఓ పరమాత్మా! నరసింహరూపా! మా సుతులు మా సుగతుల కోసం శ్రాద్ధాలు పెడుతున్నారు; పిండాలు, నువ్వులు నీళ్ళూ వదులుతున్నారు. కాని అవి మాకు అందనీయకుండా ఈ హిరణ్యాక్ష అసురుడు అమిత కోపంతో వేగంగా వచ్చి, హరించి, పట్టుకు పోయేవాడు. ఆ స్వార్థపర దుండగీడుని ఇక్కడ సంహరించావు. మా కష్టాలు తీర్చావు. నీకు నమస్కారములు”
