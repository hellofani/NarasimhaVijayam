7-168-శా.
అంధేందూదయముల్ మహాబధిర శంఖారావముల్ మూక స
ద్గ్రంథాఖ్యాపనముల్ నపుంసక వధూకాంక్షల్ కృతఘ్నావళీ
బంధుత్వంబులు భస్మ హవ్యములు లుబ్ధ ద్రవ్యముల్ క్రోడ స
ద్గంధంబుల్ హరిభక్తి వర్జితుల రిక్తవ్యర్థ సంసారముల్.
ప్రతిపదార్ధము
అంధ = గుడ్డివానిపాలిటి; ఇందు = చంద్రుని; ఉదయముల్ = ఉదయించుటలు; మహా = మిక్కిలి; బధిర =చెవిటివానిచెంత; శంఖ = శంఖముయొక్క; ఆరావముల్ = శబ్దములు; మూక = మూగవానిచేత; సత్ = మంచి; గ్రంథ = గ్రంథములను; ఆఖ్యాపనముల్ = చెప్పించుటలు; నపుంసక =మగతనములేనివాని; వధూకాంక్షల్ = మగువలపొందుకోరుటలు; కృతఘ్నా = మేలుమరచెడి; ఆవళీ =సమూహముతోటి; బంధుత్వంబులు = చుట్టరికములు; భస్మ = బూడిదలోపోసిన; హవ్యములు =హోమములు; లుబ్ద = లోభియొక్క; ద్రవ్యముల్ = సంపదలు; క్రోడ = పందికైన; సద్గంధంబుల్ = సువాసనలు; హరి = నారాయణుని; భక్తి = భక్తిని; వర్జితుల = వదలినవారి; రిక్త =శూన్యములైన; వ్యర్థ = ప్రయోజనహీనములైన; సంసారముల్ = సంసారములు.
భావము
లోకంలో గుడ్డివాడికి వెన్నెల నిరుపయోగం; చెవిటివాడికి శంఖ ధ్వని వినబడదు; మూగవాడికి గ్రంథపఠనం సాధ్యపడదు; నపుంసకుడికి కాంత మీద కోరిక ఫలించదు; కృతఘ్నుడికి బంధుత్వం కుదరదు; బూడిదలో పోసిన హోమద్రవ్యాలు నిరుపయోగమైనవి; పిసినిగొట్టు వాడికి సంపద పనికి వచ్చేది కాదు; పందికి పన్నీరు వంటి సువాసనలు తెలియనే తెలియవు; అలాగే విష్ణు భక్తి లేని వారి జీవితాలు నిస్సారములైనవి, వ్యర్థములైనవి. అని భావిస్తాను.
